  the AppStarter preload loop skips a gated route whose condition fails (`Skip optional {route}`).
- **Condition evaluator** (`util/feature.rs`): faithful port of Java `Feature.isRequired` —
  comma-separated **OR**, `!key` negation, `key=value` / `key=` / `key` forms, case-insensitive,
  **unset key never matches** (no implicit default). Made unit-testable via a lookup-closure core
  (`is_required_with`); 7 unit tests over the forms.
